### Docker Hub ()

Docker

```bash
docker run -d -p 3000:3000 --name MBTI \
  -e OPENAI_API_URL=your-api-url \
  -e OPENAI_API_KEY=your-api-key \
  -e OPENAI_MODEL=moonshotai/kimi-k2 \
  qunqin45/aurora-mbti:latest
```

